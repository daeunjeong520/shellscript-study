##### 매개변수

```
$0: 실행된 스크립트 이름
$1~${10} : 첫번째, 두번째... N번째 인자
$*: 모든 인자 (하나의 문자열로 합쳐짐)
$@: 모든 인자 (개별)
$#: 전달된 인자 개수
$$: 현재 스크립트의 PID
$?: 마지막 실행 명령의 종료 상태 코드
```

##### 매개변수 확장

```bash
# VAR가 설정되지 않음/Null일경우 default 출력만
${VAR:-default}
# VAR가 설정되지 않음/Null일경우, default로 초기화(변수 저장)
${VAR:=default}
# VAR가 존재하면 value반환, 없거나 Null이면 null출력
${VAR:+value}
# VAR가 Null이면, 에러메세지 출력 후 종료. VAR가 설정되지 않으면, 에러메세지 출력 후 종료
${VAR:?error}
# VAR가 Null이면, Null출력. VAR가 설정되지 않으면, 에러메세지 출력 후 종료
${VAR?error}
```

##### 문자열 확장

```bash
${VAR#pattern}  #(앞에서부터)처음찾은 패턴 앞 모두 제거(*필요)
${VAR##pattern} #(앞에서부터)마지막으로 찾은 패턴 앞 모두 제거(*필요)
${VAR%pattern}  #(뒤에서부터)처음찾은 패턴 뒤 모두 제거(*필요)
${VAR%%pattern} #(뒤에서부터)마지막으로찾은 패턴 뒤 모두 제거(*필요)
```

##### 문자열 치환

```bash
${VAR/찾는거/바꿀거}  #(앞에서부터)처음찾은 문자열 변경.바꿀대상 지정안하면 삭제
${VAR//찾는거/바꿀거} #문자열 전체 변경. 바꿀대상 지정안하면 삭제
${VAR/#처음/바꿀거}   #시작문자열이 패턴과 맞는 경우 변경
${VAR/%마지막/바꿀거} #마지막문자열이 패턴과 맞는 경우 변경
```
